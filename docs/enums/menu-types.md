# Growlauncher Menu Types

Menu types define the different UI components available in Growlauncher's module system.

| MenuType     | Num | Description              |
| :-           | :-: | ------------------------ |
| Toggle       | 0   | Boolean toggle switch    |
| Slider       | 1   | Numeric slider control   |
| Item_Picker  | 2   | Item selection interface |
| Json_Data    | 3   | JSON data display        |
| Module       | 4   | Module container         |
| String       | 5   | Text input field         |
| Select Tile  | 6   | Tile selection interface |
| Display List | 7   | List display component   |

## Note

Menu types are used for creating user interface elements in modules. Each type provides different input methods and visual representations.
