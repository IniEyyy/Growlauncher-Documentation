# Vectors

Vector structures for positions, sizes, and coordinates.

## Vector2

2D vector for positions and coordinates.

### Properties

| Field | Type   |
| :-    | :-:    |
| `x`   | number |
| `y`   | number |

## Vector3

3D vector for spatial coordinates.

### Properties

| Field | Type   |
| :-    | :-:    |
| `x`   | number |
| `y`   | number |
| `z`   | number |

## Vector4

4D vector commonly used for colors or RGBA values.

### Properties

| Field | Type   |
| :-    | :-:    |
| `x`   | number |
| `y`   | number |
| `z`   | number |
| `w`   | number |

## Rect

Rectangle structure for areas and bounds.

### Properties

| Field | Type   |
| :-    | :-:    |
| `x`   | number |
| `y`   | number |
| `w`   | number |
| `h`   | number |
