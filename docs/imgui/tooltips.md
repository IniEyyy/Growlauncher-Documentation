# Tooltips

Tooltip display functions for ImGui.

## Functions

---

### BeginTooltip()
```lua
BeginTooltip()
```
- **Description**: Begins tooltip
- **Returns**: None

---

### EndTooltip()
```lua
EndTooltip()
```
- **Description**: Ends tooltip
- **Returns**: None

---

### SetTooltip()
```lua
SetTooltip(fmt:string, ...:any)
```
- **Description**: Sets tooltip with formatted text
- **Parameters**: 
  - `fmt:string` - Format string
  - `...:any` - Format arguments
- **Returns**: None

---

### SetTooltip()
```lua
SetTooltip(text:string)
```
- **Description**: Sets tooltip with text
- **Parameters**: `text:string` - Tooltip text
- **Returns**: None
