# Item Utilities

Item helper functions for ImGui.

## Functions

---

### IsItemHovered()
```lua
IsItemHovered(flags:ImGuiHoveredFlags|nil) -> boolean
```
- **Description**: Checks if item is hovered
- **Parameters**: `flags:ImGuiHoveredFlags|nil` - Hover flags (optional)
- **Returns**: `boolean` - Item is hovered
